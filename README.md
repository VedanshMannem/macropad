# Setup

1) Set up [OPL Libary](https://github.com/Seeed-Studio/OPL_Kicad_Library/)
2) Add this for the [OLED footprints](https://gi1thub.com/gorbachev/KiCad-SSD1306-0.91-OLED-4pin-128x32.pretty/blob/master/SSD1306-0.91-OLED-4pin-128x32.kicad_mod)
3) Download all the project files into KiCad/Fusion or whatever system you use and you can edit however

Materials (No BOM):
- 4x Cherry MX Switches
- 6x SK6812 MINI Leds
- 1x XIAO RP2040
- 4x Blank DSA Keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset
- 1x OLED 0.91in display

First time CADding and second time doing PCBs. Learned lots about routing, edge cuts, and just general CADding.

Cool features:
- Macros for productivity (vibe coding)
- Layers for different modes to increase functionality
- Layeres: Volume/mute, productivity, WASD
- Switch between layers with 5s hold
- LED & OLED display showing different layers
